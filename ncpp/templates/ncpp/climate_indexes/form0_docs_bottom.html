About this Site

NCPP's climate indexes service is in the early stages of development. The purpose of this application is to demonstrate the capability to access climate indices across the internet from different servers. Along with the climate index data, NCPP is developing the ability to retrieve and display metadata that describes the data, and translational information that will aid the user of the data in translating the raw numbers into usable information. Please see the Climate Indexes Site for more information on the goals of this project.
